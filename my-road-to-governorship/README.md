# My Road to Governorship

This directory is the durable project workspace for Dr. Max Justice's long-term Road to Governorship initiative.

## Purpose

Use this workspace to organize research, strategy, policy development, public-service preparation, stakeholder mapping, communications, milestones, and supporting evidence related to a potential future campaign for Governor of Pennsylvania.

## Agent Handoff

Agents working in this directory should:

1. Preserve source attribution and distinguish verified facts from assumptions.
2. Record major decisions, open questions, risks, and next actions.
3. Keep research and planning nonpartisan unless a specific strategic task requires otherwise.
4. Avoid publishing personal, confidential, or campaign-sensitive information without explicit authorization.
5. Treat this README as the project entry point and add a clear index as the workspace grows.
